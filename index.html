Acto II

Érase una vez tres jóvenes visionarios con la meta de formarse como desarrolladores FullStack. Contaban con el acompañamiento de profesionales durante todo el proceso, lo cual los motivó aún más a comprometerse con las actividades abordadas a lo largo de su formación

A medida que avanzaban, los tres jóvenes se implicaron de tal manera que empezaron a desarrollar sitios web para diferentes personas, adquiriendo la experiencia necesaria para convertirse en profesionales capacitados. Su camino no fue fácil, pues estuvo lleno de altibajos, como una verdadera montaña rusa.

Finalmente, su esfuerzo y dedicación dieron fruto, ya que lograron crear una empresa que fue bien recibida por los usuarios y clientes que utilizaron sus servicios, cerrando así una historia con un final exitoso.

Acto III

Cosas que todo desarrollador web debe saber

  • Lenguajes de marcado para el frontend
  • Lenguajes de programación para el backend
  • APIs
  • Debe saber como facilitar la interacción del usuario con el frontend o interfaz de usuario
  • Debe saber como organizar su código para poder encontrar posibles errores con fácilidad

Pasos para crear tu primera página web

  1. Elige de que va a tratar tu página web
  2. Considera las partes que está tendrá y como los organizarás
  3. Crea la lógica detrás de tu página web y soluciona errores
  4. Crea tu interfaz gráfica, una que sea fácil de usar, que este unida correctamente al backend y que sea agradable visualmente
  5. Prueba las funcionalidades las veces que sea necesario
  6. No dejes de actualizarla
